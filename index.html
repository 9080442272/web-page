Quick Integration Reference

Authentication Header

Authenticate your HTTP requests by passing your API key as a Bearer token in the Authorization header:

Authorization: Bearer 8f6412c5e3969942032522cb...

Node.js Client Code

A simple snippet to make authorized requests from your Node.js application:

const axios = require('axios');
const response = await axios.post('https://api.devcore.io/v1/query', {
  headers: {
    'Authorization': `Bearer ${process.env.API_KEY}`,
    'Content-Type': 'application/json'
  },
  data: { query: 'active' }
});
